# git2

Crystal bindings to [libgit2](https://libgit2.org/libgit2/#v1.1.0)

## Installation

1. Install libgit2

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     git2:
       github: mdwagner/git2.cr
   ```

1. Run `shards install`

```crystal
require "git2"
```
